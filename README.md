📝 Project Overview
Eat All is a comprehensive food management platform. It allows users to browse a diverse menu, search for specific items, and perform administrative tasks such as adding, editing, or deleting food items in real-time. The application features a modular architecture and a modern UI that adapts to the user's theme preference.

✨ Key Features
Theme Toggle: A professional Dark/Light mode switch located in the Navbar that remembers your choice using localStorage.

Full CRUD Functionality:

Create: Add new items using Object.assign() to merge default values.

Read: View detailed object data using Object.keys(), values(), and entries().

Update: Modify item names and prices dynamically.

Delete: Remove items with instant UI updates.

Advanced Search & Filtering: Find food items instantly using 10+ string methods for data cleaning and matching (e.g., .trim(), .toLowerCase(), and .includes()).

Data Protection: Core restaurant configurations are locked using Object.freeze() to prevent accidental changes.

🛠️ Technologies Used
HTML5: Semantic structure for web accessibility.

Tailwind CSS: Utility-first framework for modern, responsive styling and Dark Mode support.

JavaScript (ES6+):

DOM Manipulation: Dynamic content rendering and event handling.

Array Methods: .map() for rendering, .filter() for searching, and .reduce() for menu analysis.

Object Methods: Extensive use of Object.assign(), Object.keys(), and Object.freeze() for data management.
